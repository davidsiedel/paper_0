\subsection{Résolution par équilibre de cellule}

We propose an alternative to the static condensation solving algorithm, postulating an implicit relation between the increment of the cell unknowns and the increment of the faces, and consisting in solving locally a nonlinear system on the cell increment with a fixed face increment, in order to check the equilibrium of the cell with its faces at each iteration of the global problem. This new solution scheme is described in Figure \ref{res_cond}, where we note $i$ a Newton iteration for solving the global problem on the set of face unknowns, and $j$ a Newton iteration for solving the local problem on the cell unknowns in an element.
